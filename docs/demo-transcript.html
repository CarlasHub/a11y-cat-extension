Dark-mode demo transcript

Transcript for the current A11Y Cat private beta walkthrough video.

  1. Open a safe demo page in Chrome desktop with dark mode enabled.
  2. Open the A11Y Cat extension panel from the toolbar.
  3. Run Scan and wait for the result summary.
  4. Open Confirmed Issues and apply severity filters: Critical, Serious, Moderate, Minor.
  5. Open a finding detail and use Highlight element.
  6. Review Manual / Incomplete / Review items and check why human judgement is required.
  7. Review Scan Limitations before deciding coverage is complete.
  8. Export CSV issue log and JSON evidence bundle.
  9. Open local data controls and show where local data can be cleared.
  10. Close with the limitation statement: A11Y Cat supports review workflows and does not replace manual accessibility testing or real assistive-technology validation.
