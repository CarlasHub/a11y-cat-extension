Documentation boundary

This site documents the A11Y Cat Chrome extension. It does not claim full WCAG conformance, accessibility certification, Chrome Web Store approval, production readiness, or replacement of expert manual and assistive technology testing.

A11Y Cat Extension is documented as a local-first accessibility inspection extension. Automated and guided review outputs support human review; they do not prove WCAG conformance.
