Troubleshooting

Common scan/runtime issues and direct actions to recover.

Extension panel does not open

  • Confirm the extension is enabled in chrome://extensions.
  • Reload the extension and refresh the page.
  • Try a standard https:// page (not browser-internal pages).

Scan shows limitations or incomplete coverage

  • Open Diagnostics and inspect runtime limitation signals.
  • Test additional interaction states manually (dialogs, tabs, menus, auth states).
  • Treat limitation output as coverage risk, not as a pass result.

Broken Links shows unverified or timeout

  • unverified means the link was not fetched (for example cross-origin).
  • timeout means verification did not complete in time.
  • Validate final behavior manually before filing as accessibility failure.

Export includes sensitive data

  • CSV/JSON may include URL/title/selectors/snippets.
  • Review and redact before sharing externally.

Need support diagnostics for issue reports

  • Use Diagnostics export/copy from the extension panel.
  • Attach the output and reproduction steps to your issue report.
