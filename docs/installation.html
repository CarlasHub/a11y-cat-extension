Installation

Install the current extension build

The documented install path is the Chrome or Chromium MV3 extension. The public site does not claim general-availability production rollout.

Current boundary: install and test the extension as a controlled beta build unless you have a separate approved release channel.

Build the extension locally

npm install
npm run build:extension
npm run check:extension

Load the unpacked extension

  1. Open chrome://extensions.
  2. Enable Developer mode.
  3. Select Load unpacked.
  4. Choose the built dist-extension/ directory.
  5. Pin the extension from the toolbar if you need repeated access during testing.

What the release build requests

activeTab

Lets the runtime inject into the current tab only after the user clicks the toolbar action.

scripting

Runs the packaged extension runtime inside the active page after explicit user action.

storage

Stores local settings, local issue state, history, review state, and related workflow data.

The public release build does not declare broad host_permissions, hosted-service permissions, analytics permissions, or remote code execution permissions.

Verify the local install

npm run build:extension:test
npm run check:extension:test
npm run test:extension:smoke

What to check after loading

  • The toolbar icon opens the extension panel on a supported page.
  • The panel renders Dashboard, Scan Results, and the documented review modules.
  • Scan, filters, exports, and local data controls are visible.
