Installation

The current supported release path is the Chrome / Chromium MV3 extension. Broad public rollout is not currently claimed.

Current boundary: the repo documents a controlled pilot, not a broad public rollout. Do not describe this as production-ready general availability.

Build the extension locally

npm install
npm run build:extension
npm run check:extension

Load the unpacked extension

  1. Open chrome://extensions.
  2. Enable Developer mode.
  3. Choose Load unpacked.
  4. Select the repository’s dist-extension/ directory.
  5. Pin the extension from the browser toolbar if you need quick access.

Permissions used by the release build

  • activeTab: inject packaged scripts only after the user clicks the toolbar action.
  • scripting: run the packaged extension runtime in the active tab.
  • storage: store local settings, workflow state, history, and review state on the current machine.
The public release build does not declare broad host_permissions, hosted-service permissions, analytics permissions, or remote code execution permissions.

Verify the local install

npm run build:extension:test
npm run check:extension:test
npm run test:extension:smoke
