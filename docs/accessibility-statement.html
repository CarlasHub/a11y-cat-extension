Accessibility statement for this documentation site

We aim for the A11Y Cat documentation site to be usable with keyboard navigation, screen readers, zoom, and small screens. This statement describes the documentation site, not the accessibility status of every scanned page or the extension’s scan output.

Scope of this statement

This statement applies to the public documentation site hosted under /docs. It does not certify accessibility of scanned target pages, and it does not certify the extension’s scan output as conformance evidence.

Target standard

This documentation site targets WCAG 2.2 AA where feasible for a static technical documentation surface.

We do not claim full conformance unless independently verified for the current published build.

Known accessibility limitations

  • Some wide technical tables require horizontal scrolling on small screens.
  • Long pages with many sections can require substantial keyboard navigation.
  • Diagram and screenshot content is accompanied by text equivalents, but detail depth can vary by section.

How this site is reviewed

  • Automated checks for broken links, heading structure, and document integrity.
  • Manual keyboard checks for skip links, focus visibility, and sidebar navigation.
  • Manual responsive checks for zoom and narrow viewport behavior.

Report an accessibility issue

Report documentation accessibility issues through GitHub Issues. Include the page URL, browser version, operating system, and reproduction steps.
