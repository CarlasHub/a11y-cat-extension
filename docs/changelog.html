Release notes and current status

This public repository does not yet maintain a separate product changelog file. This page records the current public documentation milestone and the current support boundary without overclaiming release readiness.

Current public documentation milestone

  • The public GitHub Pages site is live.
  • The canonical Extension UI feature guide is published and linked from the public site.
  • Installation, privacy, limitations, and beta-testing guidance are published as product documentation pages.

Current support boundary

  • Controlled pilot / beta on desktop Chromium: yes.
  • Broad public rollout ready: no documented claim.
  • Real assistive-technology sign-off required: yes.
  • Public package exposes local assistant or AI-assist surfaces as stable product features: no documented claim.
Important: release readiness notes are not a compliance certificate and do not replace manual accessibility validation.
