Release notes and current status

The repository does not currently maintain a separate standalone changelog file. This page summarizes the current documented release status and where release-facing notes live.

Current documented status

  • Controlled pilot on desktop Chromium: active engineering scope.
  • Broad public rollout ready: no.
  • Manual assistive-technology validation is still required before any production readiness claim.
  • The public Chrome Store package does not expose AI assistance or local assistant backend actions.

Current automated release gate

npm run verify:release
Important: the release readiness checklist is a release-discussion tool. It is not a compliance certificate.

Where release-facing notes currently live

  • Public repository README for current public repository context
  • Installation for the current install path
  • Known limitations for the current support and limitation boundary
  • Beta testing guide for the current controlled pilot workflow

Why this page exists

This site needs a stable release-notes/status page even though the repository does not currently publish a formal changelog stream. If a formal changelog is added later, this page should point to it and stop acting as a summary placeholder.
