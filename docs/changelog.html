Release notes and current status

Current public documentation milestone

  • The canonical Extension UI feature guide is published and linked from the public site.
  • Installation, privacy, limitations, and beta-testing guidance are published as user-facing product documentation.

Current release boundary

  • Stable public AI-assist or assistant surface: no published claim.
Important: release-status notes are not a compliance certificate and do not replace manual accessibility validation.
