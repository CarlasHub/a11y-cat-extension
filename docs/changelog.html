Public documentation

Changelog and release status

Release-facing notes for the public documentation boundary.

Current documented release state

The documentation describes A11Y Cat Extension as a Chromium-focused extension package unless a release note states otherwise. Do not treat this documentation as Chrome Web Store approval, production certification, or a public rollout decision.

Documentation changes in this guide refresh

  • Updated public privacy, support, permission, and limitations documentation for the current extension package.
  • Kept public navigation focused on user-facing support, privacy, feature, and limitation pages.
  • Refreshed privacy/local-data wording for Browser Preference QA Controls, dormant contrast-review state, local-first processing, and user-controlled exports.
  • Standardised scanner classification, local-first privacy language, contrast routing, design screenshot, and Screen Reader Review boundaries across the site.
  • Kept the public documentation boundary explicit: the extension supports accessibility review, but it does not certify WCAG conformance or replace manual and assistive-technology testing.

Release note requirements

Future release notes should record package version, commit, release boundary, browser support, permission changes, storage/export changes, known limitations, and whether privacy documentation was reviewed.
