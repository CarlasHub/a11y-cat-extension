Release status

Release notes and current status

This public repository does not yet publish a separate product changelog stream. This page records the current public documentation milestone and the current release boundary.

Current public documentation milestone

  • The public GitHub Pages site is live.
  • The canonical Extension UI feature guide is published and linked from the public site.
  • Installation, privacy, limitations, and beta-testing guidance are published as user-facing product documentation.

Current release boundary

  • Controlled pilot or beta on desktop Chromium: yes.
  • Broad public production rollout: no published claim.
  • Manual assistive-technology sign-off still required: yes.
  • Stable public AI-assist or assistant surface: no published claim.
Important: release-status notes are not a compliance certificate and do not replace manual accessibility validation.

Where current public notes live

  • Public repository on GitHub for the product entry point
  • Installation for the current install path
  • Privacy and local data for storage and export boundaries
  • Known limitations for support boundaries and manual-review requirements
  • Beta testing guide for the controlled pilot workflow

Next release-note improvement

If the public repository starts publishing formal product releases, this page should link to that changelog instead of acting as a status summary.
