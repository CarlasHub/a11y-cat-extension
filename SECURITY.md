# Security Policy

Report security or privacy issues through GitHub Security Advisories for this repository when available, or through the support route documented on the public support page:

https://carlashub.github.io/a11y-cat-extension/docs/support.html

Do not include private page content, customer data, authentication tokens, internal URLs, exported scan files, or screenshots containing sensitive information in a public issue.

This public repository contains documentation and store-support assets only. Security reports about the private extension implementation should include reproduction steps and affected extension version details without attaching sensitive scan evidence unless explicitly requested through a private channel.
