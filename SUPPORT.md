# Support

Use the public support page for troubleshooting guidance and contact boundaries:

https://carlashub.github.io/a11y-cat-extension/docs/support.html

Before reporting an issue, check:

- https://carlashub.github.io/a11y-cat-extension/docs/troubleshooting.html
- https://carlashub.github.io/a11y-cat-extension/docs/limitations.html
- https://carlashub.github.io/a11y-cat-extension/docs/privacy-policy.html

Do not post sensitive page content, customer data, exported reports, private URLs, or authentication details in public GitHub issues. Redact screenshots and scan evidence before sharing.
