A11Y Cat

Local-first accessibility inspection extension.

A11Y Cat is a Chrome MV3 extension for deterministic live-page accessibility inspection, structured manual-review buckets, local history, local evidence export, and explicit runtime limitation reporting.

Accessibility limitation: A11Y Cat supports accessibility review workflows. It does not replace manual testing, real assistive-technology validation, or full WCAG conformance review.

What the extension covers

axe-core scanning

Deterministic automated checks remain visible as confirmed findings with preserved source metadata.

Confirmed issues

Confirmed failure output stays separate from review buckets, advisories, and limitations.

Review items

Manual, incomplete, visual, state-limited, and human-judgement review items remain clearly classified.

Exports

CSV issue export, JSON evidence export, and local issue-state export/import are documented as local user actions.

Local history

Previous-scan comparison and retained local history are explained as implementation-backed stored state.

Language and spelling review

Language mismatch and spelling review are presented as bounded review aids, not automatic compliance verdicts.

Metadata and broken links

Open Graph, JSON-LD, Schema.org, and same-origin link verification are surfaced as separate review modules.

Diagnostics

Structured runtime diagnostics, support JSON, and explicit limitation reporting remain visible to users.

Screen reader review aid

The virtual review surface supports QA workflows without pretending to replace real NVDA, JAWS, or VoiceOver testing.
