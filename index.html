A11Y Cat

Local-first accessibility inspection extension.

A11Y Cat is a local-first accessibility inspection extension for developers, QA teams, and accessibility reviewers.

It separates confirmed automated findings from review items, advisory notes, and scan limitations. Results are labelled by source and confidence so teams can see what was detected automatically and what still needs human judgement.

Support boundary: A11Y Cat supports accessibility review workflows. It does not replace manual testing, real assistive-technology validation, or full WCAG conformance review.

What the extension covers

axe-core scanning

Preserves rule provenance and separates direct engine output from product-owned review logic.

Confirmed issues

Keeps confirmed automated findings separate from review-only and advisory output.

Review items

Shows incomplete, visual, state-limited, and human-judgement findings without overstating certainty.

Exports

Provides CSV, JSON evidence, local issue-state, diagnostics, and virtual review exports as explicit user actions.

Local history

Stores previous scans locally for comparison on the same origin and path.

Language and spelling review

Flags language mismatch and spelling review output without treating them as automatic compliance failures.

Metadata and broken links

Exposes metadata review and broken-link verification as separate modules with documented boundaries.

Diagnostics

Shows structured runtime signals, support data, and explicit scan limitations.

Screen Reader Review aid

Provides a virtual review aid for QA workflows. It is not a substitute for VoiceOver, NVDA, or JAWS testing.
